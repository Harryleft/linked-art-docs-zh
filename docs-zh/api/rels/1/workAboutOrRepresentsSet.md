## 作品AboutOrRepresentsSet

返回 作品s which are 关于 or represent the 集合.

### 示例

从...的记录中 the 集合 for the Archives of O. C. Marsh, the records for an article 关于 the archive and an image of the archive 将在响应中

* 给定类: Set
* 返回类: Work
* 关系: 关于 / represents
