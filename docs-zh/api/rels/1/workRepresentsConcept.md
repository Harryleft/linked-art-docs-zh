## 作品RepresentsConcept

返回 visual 作品 represent or depict the 概念, or an instance of the 概念

参见相关的 [模型文档](/model/对象/关于ness/#depiction)

### 示例

从...的记录中 the 概念 of Dogs, 的记录 the 作品 of the Night Watch 将在响应中 (as the image depicts an unidentified dog)


### 详情

* 给定类: Concept
* 返回类: Work
* 关系: represents
