## 作品MemberOfSet

Return 作品s which are members of the 集合.

参见相关的 [模型文档](/model/collection/#features)

### 示例

从...的记录中 the 集合 of 作品s which describe the Night Watch, 的记录 Bikker's "The Night Watch" 将在响应中


### 详情

* 给定类: Set
* 返回类: Work
